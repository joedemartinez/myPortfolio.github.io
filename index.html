Membership

This is a web application that allows users to create and manage groups, register members, assign them to different membership levels and access exclusive content or features based on their membership level. This project was developed using PHP, MySQL, HTML, CSS, and JavaScript.

Key features include:

  • Group creation and management: Admins should be able to create and manage groups, and assign different membership levels to members.
  • Registration and login: Members should be able to register for the group by providing their personal information, and then login securely to access the app's features and content.
  • Membership levels: The app should support multiple membership levels, each with its own set of features and benefits.
  • Content access: Members should be able to access exclusive content or features based on their membership level.
  • Notifications: The app should have a notification system to inform members about new content, membership updates, or other important information.
  • Dues and Donation Payment: Dues and donation collection functionality to enable members to pay their dues or make donations

The project follows a Model-View-Controller (MVC) design pattern and was built using the CodeIgniter PHP framework.

Daily Tasks

The Daily Task is a web-based platform that allows users to create and manage daily tasks and to-do lists. This project was developed using PHP, MySQL, HTML, CSS, and JavaScript.

The Daily Tasks features include:

  • User registration and login
  • Task creation and management with due dates and priority levels
  • Task filtering and sorting for upcoming due dates and task completion

The project follows a Model-View-Controller (MVC) design pattern and was built using the CodeIgniter PHP framework.

The app was designed to address the common challenge in IT support of managing multiple tasks, keeping track of the status of each task, and ensuring that no task falls through the cracks. With the IT Support Daily Task, IT support professionals can easily create and manage their daily tasks, set due dates, and prioritize them based on urgency.

In addition, the app enables IT support professionals to collaborate and share task information with their team members, allowing for better coordination and communication.
